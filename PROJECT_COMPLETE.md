## 🎮 מדריך שימוש מהיר

### 1️⃣ הקלטה עם אפקטים בזמן אמת:

```
1. 🎙️ לחץ "התחל הקלטה"
2. 🎧 לחץ "ניטור חי" (חבר אוזניות!)
3. 🎚️ גרור סליידרים בזמן שאתה שר:
   - Reverb 60% → הד של אולם!
   - Echo 30% → הד חוזר!
   - Pitch +2 → צליל גבוה יותר!
4. ⏹️ עצור כשסיימת
5. ▶️ השמע את התוצאה
6. 💾 ייצא MP3
```

### 2️⃣ שימוש ב-Presets:

```
לחץ על אחד:
🏛️ אולם קונצרטים - Reverb 70%
🎙️ אולפן - מאוזן
📢 אקו כבד - Echo 80%
🎛️ בס מוגבר - Bass +10 dB
```

### 3️⃣ מיקס עם קריוקי:

```
1. 📁 טען קריוקי
2. 🎤 הקלט ווקאל (עם אפקטים!)
3. 🎚️ התאם עוצמות (פלייבק 50%, ווקאל 80%)
4. 🔀 לחץ "ערבב"
5. 💾 ייצא תוצאה סופית
```

💎 Pitch ±3 = שינוי טבעי
💎 Bass +5 dB = צליל מלא
💎 Treble +3 dB = בהירות
